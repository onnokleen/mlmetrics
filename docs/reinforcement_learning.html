    16 Reinforcement Learning

    16.1 Summary

    Reinforcement learning provides a powerful framework for solving sequential decision-making problems under uncertainty. Key concepts include:

    • The agent-environment loop, where an agent takes actions, receives rewards, and observes new states.
    • The Markov Decision Process (MDP) as the mathematical foundation.
    • The Bellman equation, which provides a recursive definition of value.
    • Q-learning, a model-free algorithm that directly learns the optimal action-value function, \(Q^*(s,a)\).

    The exploration-exploitation trade-off is a central challenge, addressed by strategies like \(\epsilon\)-greedy and optimistic initialization. While we have focused on tabular methods, modern deep reinforcement learning extends these ideas to complex, high-dimensional problems by using neural networks to approximate value functions or policies.

    16.2 Exercises

    Exercise 16.1: Markov Decision Processes

    Interpret the MDP as a simple capital/inventory problem with states \(S=\{0,1,2,3,4\}\) (stock level). In each period the decision-maker chooses:

    • Action A (“harvest/liquidate”): In any state \(s\), receive reward \(s\) and reset to state \(0\) with probability 1.
    • Action B (“continue/accumulate”): In any state \(s\), receive reward \(0\) and transition to \((s+1)\!\!\pmod 5\) with probability \(p = 0.8\) and to \(0\) with probability \(1-0.8 = 0.2\).

    The goal is to maximize the expected discounted reward with discount factor \(\gamma\).

    1. Let policy \(\pi\) be “take action B for all \(s \ge 2\) and action A for \(s \in \{0,1\}\)”. Write and solve the Bellman equations to find the state-value function \(v_\pi(s)\) for all states under this policy for:
      1. \(\gamma=0\)
      2. \(\gamma=0.9\)
    2. Find the optimal policy \(\pi^*\) and the optimal state-value function \(v_*(s)\) for a myopic agent with \(\gamma=0\).
    3. Now, consider a non-myopic agent with \(\gamma=0.9\). Can you find the optimal policy for state \(s=4\) without solving the full system of Bellman equations? This can be done via comparing the best possible outcome of taking action B from state 4 to the guaranteed outcome of taking action A; that is, comparing \[Q_*(s,a) = R(s,a) + \gamma \sum_{s'} \mathbb P_\pi (s'|s,a) v_*(s')\]

    for \(s = 4\) and \(a = A, B\) where \(\mathbb P_\pi (s'|s,a)\) refers to the probability of going from state \(s\) to state \(s'\) after taking action \(a\) under policy \(\pi\).

    Part 1 and 2 are exam level, part 3 is slightly too difficult for an exam

    Hint for Part 1

    For policy evaluation, use the Bellman expectation equation for the given policy \(\pi\): \(v_\pi(s) = R(s, \pi(s)) + \gamma \sum_{s'} P(s'|s, \pi(s)) v_\pi(s')\).

    When \(\gamma=0\), the equation simplifies significantly as only the immediate reward matters, so \(v_\pi(s) = R(s, \pi(s))\).

    For \(\gamma=0.9\), you will have a system of linear equations. Start by evaluating states whose values can be determined directly (like states 0 and 1), then work backward through states 4, 3, and 2.

    Hint for Part 2

    When \(\gamma=0\), the agent is completely “myopic” and only cares about the immediate reward. The Bellman optimality equation simplifies to:

    \[v_*(s) = \max_{a} R(s,a)\]

    Compare the immediate rewards of actions A and B in each state.

    Hint for Part 3

    To find the optimal action in state 4, compare the optimal action-values \(Q_*(4, A)\) and \(Q_*(4, B)\).

    For action A from state 4:

    \[Q_*(4, A) = 4 + 0.9 \cdot v_*(0)\]

    For action B from state 4:

    \[Q_*(4, B) = 0 + 0.9 \cdot [0.8 \cdot v_*(0) + 0.2 \cdot v_*(0)] = 0.9 \cdot v_*(0)\]

    Notice that both actions lead to state 0 in expectation, but action A provides an additional immediate reward of 4. Can you conclude which action is optimal without knowing \(v_*(0)\)?

    Solution

    1. Policy Evaluation for \(\pi\):

      1. Case \(\gamma=0\): With no discounting of the future, the value of a state under a policy is just the immediate reward.
      • \(v_\pi(0)\): Policy \(\pi\) takes action A, reward is \(s=0\). So, \(v_\pi(0) = 0\).
      • \(v_\pi(1)\): Policy \(\pi\) takes action A, reward is \(s=1\). So, \(v_\pi(1) = 1\).
      • \(v_\pi(2), v_\pi(3), v_\pi(4)\): Policy \(\pi\) takes action B, reward is \(0\). So, \(v_\pi(2)=v_\pi(3)=v_\pi(4)=0\). The state-value function is \(v_\pi = [0, 1, 0, 0, 0]\).
      1. Case \(\gamma=0.9\): We solve the system of Bellman equations for policy \(\pi\).
      • For \(s=0\) (Action A): \(v_\pi(0) = 0 + 0.9 \cdot v_\pi(0) \implies v_\pi(0) = 0\).
      • For \(s=1\) (Action A): \(v_\pi(1) = 1 + 0.9 \cdot v_\pi(0) = 1 + 0 = 1\).
      • For \(s=4\) (Action B): \(v_\pi(4) = 0 + 0.9 \cdot [0.8 \cdot v_\pi((4+1)\pmod 5) + 0.2 \cdot v_\pi(0)] = 0.9 \cdot [0.8 \cdot v_\pi(0) + 0.2 \cdot v_\pi(0)] = 0.9 \cdot v_\pi(0) = 0\).
      • For \(s=3\) (Action B): \(v_\pi(3) = 0 + 0.9 \cdot [0.8 \cdot v_\pi(4) + 0.2 \cdot v_\pi(0)] = 0.9 \cdot [0.8 \cdot 0 + 0.2 \cdot 0] = 0\).
      • For \(s=2\) (Action B): \(v_\pi(2) = 0 + 0.9 \cdot [0.8 \cdot v_\pi(3) + 0.2 \cdot v_\pi(0)] = 0.9 \cdot [0.8 \cdot 0 + 0.2 \cdot 0] = 0\).

      Even with \(\gamma=0.9\), the state-value function is \(v_\pi = [0, 1, 0, 0, 0]\). This is because action B gives 0 immediate reward and only leads to states that themselves have a value of 0 under this policy.

    2. Optimal Policy for \(\gamma=0\): When \(\gamma=0\), the agent only seeks to maximize its immediate reward. We compare the rewards for actions A and B in each state:

      State \(s\) Reward(A) Reward(B) Optimal Action
      0 0 0 A or B
      1 1 0 A
      2 2 0 A
      3 3 0 A
      4 4 0 A

      The optimal policy is to always take action A for all states \(s\) (with action B also being optimal at \(s=0\) since both actions yield the same reward). The optimal state-value function \(v_*(s)\) is the reward from taking action A, so \(v_*(s) = s\), or in vector form: \(v_* = [0, 1, 2, 3, 4]\).

    3. Optimal Policy for \(s=4\) with \(\gamma=0.9\): Yes, we can determine the optimal action for \(s=4\) by comparing the optimal action-value function \(Q_*(4, A)\) and \(Q_*(4, B)\), which represent the expected discounted return from taking each action and then following the optimal policy thereafter.

      Using the Bellman equation for action-values:

      • Action A: \(Q_*(4, A) = R(4,A) + \gamma \cdot v_*(0) = 4 + 0.9 \cdot v_*(0)\)
      • Action B: From state 4, action B transitions to state \((4+1) \bmod 5 = 0\) with probability 0.8 and to state 0 with probability 0.2. Thus: \[Q_*(4, B) = 0 + 0.9 \cdot [0.8 \cdot v_*(0) + 0.2 \cdot v_*(0)] = 0.9 \cdot v_*(0)\]

      Comparing these:

      \[Q_*(4, A) - Q_*(4, B) = [4 + 0.9 \cdot v_*(0)] - [0.9 \cdot v_*(0)] = 4 > 0\]

      Since \(Q_*(4, A) > Q_*(4, B)\) regardless of the value of \(v_*(0)\), the optimal action is \(\pi^*(4) = A\).

      Economic interpretation: Action A yields an immediate reward of 4 and transitions to state 0. Action B yields no immediate reward and, in expectation, also transitions to state 0 (either directly or after one additional step through state 0 itself). Since both actions lead to the same expected future state but action A provides a strictly positive immediate reward, it dominates action B. This is an example of action dominance: we can determine the optimal action without solving for the complete optimal value function.

    Exercise 16.2: Q-Learning Update

    Consider the following table of current Q-value estimates \(Q(s,a)\):

    State 1 State 2
    Action 1 1.5 2.5
    Action 2 4.0 3.0

    The agent uses a learning rate \(\alpha=0.1\) and a discount factor \(\gamma=0.5\). We observe the transition \((s,a,r,s')=(1,1,5,2)\); that is, from state \(s=1\), it takes action \(a=1\), receives a reward \(r=5\), and transitions to a new state \(s'=2\).

    1. Which entry in the Q-table gets updated, and what is its new value?
    2. What is a potential problem with a purely greedy Q-learning agent (one that never explores and always chooses the action with the highest current Q-value)?
    3. Describe two common methods to encourage exploration in Q-learning.

    Exam level

    Hint for Part 1

    Recall the Q-learning update rule: \(Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \left[ R_{t+1} + \gamma \max_{a'} Q(S_{t+1}, a') - Q(S_t, A_t) \right]\) The term \(\max_{a'} Q(S_{t+1}, a')\) is the agent’s estimate of the best possible value it can get from its new state, \(S_{t+1}\).

    Hint for Part 2

    Think about what happens if the agent’s initial Q-values are misleading. If an optimal action has a low initial Q-value by chance, will a purely greedy agent ever try it?

    Hint for Part 3

    One method involves adding randomness to the action selection policy. Another involves changing how the Q-table is initialized to incentivize trying unknown actions.

    Solution

    1. Q-table Update: The entry for the state-action pair that was observed, \(Q(s=1, a=1)\), gets updated.

      We use the Q-learning update rule:

      \[ Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \left[ R_{t+1} + \gamma \max_{a'} Q(S_{t+1}, a') - Q(S_t, A_t) \right] \]

      Plugging in the values:

      • \(Q(S_t, A_t) = Q(1, 1) = 1.5\)
      • \(\alpha = 0.1\)
      • \(R_{t+1} = 5\)
      • \(\gamma = 0.5\)
      • \(\max_{a'} Q(S_{t+1}, a') = \max_{a'} Q(2, a') = \max(2.5, 3.0) = 3.0\)

      The calculation is:

      \[ \begin{aligned} Q(1,1) &\leftarrow 1.5 + 0.1 \left[ 5 + 0.5 \times 3.0 - 1.5 \right] \\ &= 1.5 + 0.1 \left[ 5 + 1.5 - 1.5 \right] \\ &= 1.5 + 0.1 \times 5.0 \\ &= 1.5 + 0.5 = \mathbf{2.0} \end{aligned} \]

      The new value for \(Q(1,1)\) is 2.0.

    2. Problem with Pure Exploitation: A purely greedy agent is at high risk of getting stuck in a suboptimal policy. If an action that is truly optimal happens to have a low initial Q-value (due to random initialization or misleading early experience), the greedy agent may never select it. It will keep exploiting what it thinks is the best action, without ever discovering its mistake. This prevents it from finding the true optimal policy.

    3. Exploration Strategies:

      • \(\epsilon\)-greedy: This is the most common strategy. With a small probability \(\epsilon\) (e.g., \(\epsilon=0.1\)), the agent chooses a random action, regardless of its Q-values. With probability \(1-\epsilon\), it chooses the greedy action (the one with the highest Q-value). This ensures that, over time, all actions are eventually tried, allowing the agent to escape suboptimal policies.
      • Optimistic Initialization: Instead of initializing the Q-table with zeros, initialize all Q-values to a high, “optimistic” value (e.g., a value higher than any possible reward). When the agent tries an action, the update rule will almost certainly lower its Q-value, as the received reward will be less than the initial optimistic estimate. This “disappointment” encourages the agent to try all other actions at least once, as their initial high values are more attractive than the now-reduced value of the action it just tried.
